Chapter 1: Introduction to CSS

What is CSS and Why Learn It?

CSS (Cascading Style Sheets) is a stylesheet language used to control the look and feel of a webpage. While HTML is used to structure the content of a webpage, CSS is used to style it, such as changing colors, fonts, layouts, and backgrounds.

Why learn CSS? Because it allows us to make websites visually attractive, user-friendly, and responsive across different devices. It helps separate content from design, making the code easier to maintain and scale.

CSS Selectors

A CSS Selector is used to select an HTML element(s) for styling.

Example

body {
    color: red;           /* Declaration (property: value) */
    background: pink;     /* Declaration */
}

#body is the selector.

Element Selector

It is used to select an element based on the tag name.

h2 {
    color: blue;
}

Id Selector

It is used to select an element with a given id. # is used to target by id.

#first {
    color: white;
    background: black;
}

Class Selector

It is used to select an element with a given class.

.red {
    background: red;
}

Important Notes

We can group selectors like this:
h1, h2, h3, div {
    color: blue;   /* h1, h2, h3 and div will be blue */
}
We can use element class as a selector like this:
p.red {
    color: red;   /* all paragraphs will get color of red */
}
'*' can be used as a universal selector to select all the elements:
* {
    margin: 0;
    padding: 0;
}

Ways to Add CSS in HTML

Below are the three main methods to add CSS in HTML:

1. Inline CSS

  • Written inside the style attribute of an element.
  • Advantages: Quick, no external file.
  • Disadvantages: Not reusable, hard to maintain.
<h1 style="color: red; font-size: 20px;">Hello World</h1>

2. Internal CSS

  • Written inside a <style> tag within the <head>.
  • Advantages: Good for single-page sites.
  • Disadvantages: Not reusable across pages.
<style>
  h1 { color: blue; }
</style>

3. External CSS

  • Written in a separate .css file and linked with <link>.
  • Advantages: Reusable, scalable, clean.
  • Disadvantages: Needs extra file request.
<link rel="stylesheet" href="styles.css">
/* styles.css */
h1 { color: green; }

Quick Comparison Table

MethodWhere it's writtenBest ForDrawback
Inline CSSstyle attributeQuick fixesHard to maintain
Internal CSS<style> in <head>Small single-page sitesNot reusable
External CSSSeparate .css fileLarge, multi-page sitesExtra file load

Conclusion:

  • Use Inline CSS for very small fixes.
  • Use Internal CSS for single-page projects.
  • Use External CSS for professional, large projects.

Note: An inline style will override external and internal styles.

Comments in CSS

Comments in CSS are the text which is not parsed and is thus ignored.
