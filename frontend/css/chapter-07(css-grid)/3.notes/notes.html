Chapter-07 – CSS Grid

What is CSS Grid?

CSS Grid is a two-dimensional layout system used to design complex layouts using rows and columns.

  • Controls layout in rows and columns
  • Best for full-page layouts
  • More powerful than Flexbox for structure

Two-Dimensional Layout (Grid)

CSS Grid works in two directions at the same time: rows (horizontal) and columns (vertical).

Figure: Two-Dimensional Layout in CSS Grid

Grid Container & Grid Items

When display: grid is applied to an element:

  • The element becomes a grid container
  • Its direct children become grid items
.container {
  display: grid;
}
Note: Only direct children become grid items.

Grid Rows & Columns

Grid layout is created using rows and columns. 

.container {
  grid-template-columns: 200px 200px 200px;
  grid-template-rows: 100px 100px;
}
This creates 3 columns and 2 rows.

Grid Container Properties

Grid container properties are applied to the parent element. They define the grid structure and alignment.

Grid container properties:
  1. grid-template-columns
  2. grid-template-rows
  3. gap
  4. justify-items
  5. align-items

grid-template-columns

Defines the number and size of columns.

.container {
  grid-template-columns: 1fr 2fr 1fr;
}
fr means fraction of available space.

grid-template-rows

Defines the height of rows.

.container {
  grid-template-rows: 100px auto;
}

gap Property

Controls spacing between rows and columns.

.container {
  gap: 20px;
}
  • row-gap
  • column-gap
  • gap (shorthand)

justify-items

Aligns grid items horizontally inside cells.

.container {
  justify-items: center;
}
  • start
  • center
  • end
  • stretch (default)

align-items

Aligns grid items vertically inside cells.

.container {
  align-items: center;
}

Grid Item Properties

Grid item properties are applied to individual grid items. They control position and alignment.

Grid item properties:
  1. grid-column
  2. grid-row
  3. justify-self
  4. align-self

grid-column

Defines how many columns an item spans.

.item {
  grid-column: 1 / 3;
}
Item spans from column 1 to column 3.

grid-row

Defines how many rows an item spans.

.item {
  grid-row: 1 / 3;
}

justify-self

Aligns a single grid item horizontally.

.item {
  justify-self: center;
}

align-self

Aligns a single grid item vertically.

.item {
  align-self: end;
}

CSS Grid vs Flexbox

  • Grid → Two-dimensional layout (rows + columns)
  • Flexbox → One-dimensional layout
  • Grid is best for page layout
  • Flexbox is best for components

Short Exam Definition

CSS Grid is a two-dimensional layout system used to create complex and responsive layouts using rows and columns.
