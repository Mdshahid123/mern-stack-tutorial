HTML Attributes

Definition:

HTML attributes provide additional information about HTML elements. Attributes are always written in name="value" format. 
<p align="center" > Hello World </p>

ID Property

ID is used to give a unique name to an element. 
<h1 id="mainHeading" > Welcome </h1>

Anchor Tag

Definition:

The <a> tag in HTML is called the anchor tag. It is used to create hyperlinks that let you jump to another external website, webpage, section, email, or even download a file. Anchor tag is used to create links in HTML.

Basic Syntax

<a href="https://www.example.com" > Click Here </a>

Attributes

href → Attribute that tells where the link should go.

target="_blank" → Opens external link in new tab.

Practical Examples

1) Opening External Website in Same Tab

<a id="section1" href="https://www.flipkart.com/" > go to flipkart</a>

2) Link to External Website and Open in New Tab

<ahref="https://www.google.com"target="_blank" > go to google</a>

3) Download a File

<a href="./assets/file.pdf" > download notes </a>

4) Send an Email

<a href="mailto:shahiddelhi989@gmail.com" > Send Email </a>

5) Link to Section on Same Page

<p id="section1" > section 1 </p>
<a href="#section1" > go to section1 </a>

6) Link to Another Page in Same Website

<a href="./contact_page.html" > contact page </a>

Big and Small Tag

Definition:

The <big> tag is used to increase the text size and the <small> tag is used to decrease the text size.

<p> This is normal text </p>
<big> This is big text </big>
<small> This is small text </small>

Pre Tag

Definition:

The <pre> tag is used to display preformatted text. It preserves spaces and line breaks exactly as written in the HTML code. Text inside <pre> tag keeps original spacing and formatting.

Example:

<!-- Pre tag preserves spaces and line breaks -->
<pre>
To,
The Principal ABC School
Respected Sir, 
I beg to say that...
</pre>

Video Tag

Definition:

The <video> tag is used to add a video on a webpage.

Attributes Used in Video Tag:

1) src → Used to specify the video URL or file path.  
2) controls → Adds built-in video controls. Without controls, the video is visible but you cannot play, pause, or adjust volume. With controls, the browser shows toolbar (play, pause, volume, fullscreen, etc.).  
3) autoplay → Plays the video automatically (usually muted by default).  
4) loop → Repeats the video automatically.  
5) width → Sets the width of the video.  
6) height → Sets the height of the video.

Example:

<!-- Video tag with common attributes -->
<video src="video.mp4" controls autoplay loop width="400" height="300" ></video>

Heading Tag

Definition:

HTML heading tags are used to define headings or titles on a webpage. Headings help organize content and make the webpage easier to read.

Types of Heading Tags:

HTML provides six levels of headings.

< h1 >: Largest heading
< h2 >: Second largest heading
< h3 >: Medium heading
< h4 >: Smaller heading
< h5 >: Very small heading
< h6 >: Smallest heading

Example:

<!-- Example of heading tags -->

<h1> This is Heading 1 </h1>
<h2> This is Heading 2 </h2>
<h3> This is Heading 3 </h3>
<h4> This is Heading 4 </h4>
<h5> This is Heading 5 </h5>
<h6> This is Heading 6 </h6>
