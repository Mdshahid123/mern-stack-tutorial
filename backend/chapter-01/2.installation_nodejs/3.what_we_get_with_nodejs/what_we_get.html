1) Node.js Runtime:A JavaScript runtime that allows you to run JavaScript code outside the browser. It includes the V8 engine (same as Chrome) to execute JavaScript code. Enables backend development with JavaScript (e.g., using Express.js). 2) npm (Node Package Manager ):A package manager for JavaScript. Helps install, manage, and update libraries/frameworks like React, Express, etc. 4) npx (Node Package eXecute):A command-line tool that allows you to run Node.js packages without installing them globally. 4) Core Modules (Built-in Libraries):Node.js includes built-in modules called core module like:fs(file system),http,https,path,path.os,events,crypto and url no installation needed directly available for use without npm install. 5) Node REPL (Read-Eval-Print Loop):An interactive shell to run JavaScript code line by line. Open it by typing command:node
