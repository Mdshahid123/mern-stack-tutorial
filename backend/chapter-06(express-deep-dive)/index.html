What is Data Science?

Data Science is the field of study that combines data collection, processing, analysis, and visualization to extract insights and support decision-making.

  • Analyze structured and unstructured data
  • Make predictions and classifications
  • Automate decisions and generate insights
  • Use AI, ML, and statistical techniques

Branches of AI Used in Data Science

  • Machine Learning: Prediction, classification, clustering
  • Deep Learning: Complex data, images, audio, large-scale datasets
  • Generative AI: Automating reports, content, synthetic data
  • Natural Language Processing (NLP): Text and language data
  • Computer Vision (CV): Image and video data

Technologies Used in Data Science

  • Programming Languages: Python, R, SQL
  • Data Processing: Pandas, Numpy, Spark
  • Machine Learning / AI: Scikit-Learn, TensorFlow, PyTorch, Keras
  • Data Visualization: Matplotlib, Seaborn, Tableau, PowerBI
  • Big Data: Hadoop, Apache Spark
  • Cloud Platforms: AWS, GCP, Azure

Types of AI Models in Data Science

As a Data Scientist, the type of AI model is chosen based on business problem, data type, and goal.

Prediction Models

  • Goal: Forecast numeric outcomes
  • Sales prediction, Stock price forecasting, Weather prediction
  • AI Branch: Machine Learning → Regression / Deep Learning

Classification Models

  • Goal: Categorize data
  • Loan approval, Disease detection, Customer churn, Spam email detection
  • AI Branch: Machine Learning → Classification

Recommendation Models

  • Goal: Suggest products / content to users
  • Netflix, Amazon, YouTube recommendations
  • AI Branch: Machine Learning / Deep Learning

Detection / Anomaly Models

  • Goal: Detect abnormal or fraudulent activities
  • Credit card fraud detection, Intrusion detection
  • AI Branch: Machine Learning → Anomaly Detection

Clustering Models

  • Goal: Group similar items or users
  • Customer segmentation, Market analysis, Behavior analysis
  • AI Branch: Machine Learning → Unsupervised Learning

NLP Models

  • Goal: Analyze and interpret text
  • Sentiment analysis, Email classification, Resume screening, Chat analysis
  • AI Branch: NLP → ML / Deep Learning

Computer Vision Models

  • Goal: Analyze images and videos
  • Face recognition, Medical image diagnosis, Object detection
  • AI Branch: Computer Vision → Deep Learning

Generative AI Models

  • Goal: Create new content / automate insights
  • Text generation (chatbots), Image generation, Report summarization, Synthetic data
  • AI Branch: Deep Learning → Generative AI
