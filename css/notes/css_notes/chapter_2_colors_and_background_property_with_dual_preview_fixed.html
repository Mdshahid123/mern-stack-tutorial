The color property

The CSS color property can be used to set the text color of an HTML element.

Ways to specify colors in CSS

Many ways to specify colors in CSS:

1) By Color Name

p {
  color: red; /* Text color will be changed to red */
}

Similarly, we can set color for different elements.

2) HEX Code

A hex code (short for hexadecimal code) is a way to represent colors in web design and digital graphics using a 6-digit combination of numbers and letters.

  • Format: #RRGGBB
  • RR ‚Üí Red value (00‚ÄìFF)
  • GG ‚Üí Green value (00‚ÄìFF)
  • BB ‚Üí Blue value (00‚ÄìFF)

Examples:

  • #FF0000 ‚Üí Red
  • #00FF00 ‚Üí Green
  • #0000FF ‚Üí Blue
  • #FFFFFF ‚Üí White
  • #000000 ‚Üí Black

Shorthand:

  • #FFF = #FFFFFF
  • #000 = #000000

3) RGB

The RGB color model specifies colors using the intensity of Red, Green, and Blue on a scale of 0‚Äì255.

p {
  color: rgb(255, 0, 0); /* Red */
  color: rgb(0, 255, 0); /* Green */
  color: rgb(0, 0, 255); /* Blue */
}

4) RGBA

RGBA is an extension of RGB with an additional Alpha (opacity) channel, ranging from 0 (transparent) to 1 (opaque).

p {
  color: rgba(255, 0, 0, 0.5); /* Semi-transparent red */
}

5) HSL

HSL stands for Hue, Saturation, and Lightness:

  • Hue: degree on the color wheel (0‚Äì360)
  • Saturation: intensity (0%‚Äì100%)
  • Lightness: brightness (0%‚Äì100%)
p {
  color: hsl(0, 100%, 50%); /* Red */
  color: hsl(120, 100%, 50%); /* Green */
  color: hsl(240, 100%, 50%); /* Blue */
}

6) HSLA

HSLA adds an Alpha (opacity) channel to HSL.

p {
  color: hsla(0, 100%, 50%, 0.5); /* Semi-transparent red */
}

Background Properties

The background-color property

The CSS background-color property specifies the background color of a container.

.brown {
  background-color: brown;
}

The background-image property

Used to set an image as the background.

body {
  background-image: url("harry.jpg");
}

The image is by default repeated in X & Y directions.

The background-repeat property

Defines how a background image is repeated.

  • repeat-x: repeat in the horizontal direction
  • repeat-y: repeat in the vertical direction
  • no-repeat: image not repeated
  • repeat: default (repeats both directions)

The background-size property

Specifies the size of the background image.

  • cover: fits ,no empty space remains and croping and streching poassible
  • contain: image is fully visible and vacant space can be remain
  • auto: display in original size
  • <width>: set width, height auto
  • <width> <height>: set width & height

Note: Always check MDN docs to dissect a given CSS property. Practice will make you perfect.

The background-position property

Sets the starting position of a background image.

.div1 {
  background-position: left top;
}

The background-attachment property

Defines whether the background is scrollable or fixed.

.div2 {
  background-attachment: fixed;
}

The background shorthand

A single property to set multiple background values at once.

.div3 {
  background: red url("img.png") no-repeat fixed right top;
}

üëâ One or more values can be skipped, as long as the others are in order.
